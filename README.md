# 🍺 DrinkBuddy - Приложение за намиране на партньор за пиене

Приложение в стил Tinder за намиране на хора наблизо, които искат да се срещнат за по едно питие!

## ✨ Функционалности

- **Swipe карти** - Swipe наляво или надясно за избор на потенциални партньори за пиене
- **Информация за дистанция** - Вижте колко далеч е всеки потребител (в километри)
- **Любима напитка** - Всеки профил показва предпочитаната напитка на потребителя
- **Match система** - Когато двама потребители се харесат взаимно, получават match
- **Изглед на съвпадения** - Вижте всички ваши matches на едно място
- **Анимации** - Плавни swipe анимации и преходи

## 🚀 Как да стартирате

### Инсталация

```bash
npm install
```

### Стартиране на development сървър

```bash
npm run dev
```

Приложението ще бъде достъпно на `http://localhost:5173`

### Build за production

```bash
npm run build
```

## 🛠️ Технологии

- **React** - UI библиотека
- **Vite** - Build tool
- **TailwindCSS** - Styling
- **Framer Motion** - Анимации
- **Lucide React** - Икони

## 📱 Функции

### Главен екран (Карти)
- Разглеждане на профили чрез swipe
- Показване на име, възраст, дистанция и предпочитана напитка
- Swipe бутони за лесна навигация
- Undo функционалност

### Екран за Matches
- Списък с всички взаимни харесвания
- Бърз достъп до чат (UI готов)

### Профил
- Вашият профил
- Редактиране на информация
- Настройки

## 🎨 Персонализация

Можете да добавите нови профили в `src/data/profiles.js`:

```javascript
{
  id: 7,
  name: "Име",
  age: 25,
  distance: 1.0,
  drink: "Любима напитка",
  bio: "Кратка биография",
  image: "URL към снимка",
  interests: ["Интерес 1", "Интерес 2"]
}
```

## 📝 Бележки

- Приложението използва примерни данни
- Match системата работи с 50% вероятност за демонстрация
